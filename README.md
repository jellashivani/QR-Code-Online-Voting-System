# QR Code Based Secure Online Voting System

## 📌 Project Description

The **QR Code Based Secure Online Voting System** is a web-based application designed to provide a secure and efficient digital voting process. The system uses **QR code authentication** to verify voters and ensure that each voter can vote only once.

This project was developed as a **Major Project for the Bachelor of Technology (B.Tech) in Information Technology (2025)**.

The system improves the traditional voting process by reducing manual work, preventing duplicate voting, and enhancing security using QR code verification.

---

## 🎯 Objectives

* To develop a secure online voting system.
* To authenticate voters using QR code technology.
* To prevent duplicate or unauthorized voting.
* To provide an easy and efficient digital voting platform.

## ▶️ How to Run the Project

1. Install **Apache NetBeans IDE**
2. Install **MySQL Database**
3. Import the project into NetBeans
4. Configure the database connection
5. Run the project using NetBeans
6. Open the application in your browser (localhost)
